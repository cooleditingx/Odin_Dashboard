## üìñ Description

This project is a static front-end application representing a complete dashboard user interface. It was developed to practice and demonstrate advanced CSS skills, particularly **CSS Grid** and **Flexbox**, to create a complex, modern, and highly responsive web layout.

The dashboard features a persistent sidebar, a navigation header, and a main content area populated with project and activity cards. It showcases attention to design, typography, and clean, modular HTML/CSS structure.

***

## üíª Local Setup & Usage

Since this is a static website built solely with HTML and CSS, no complex build tools or server environment are required.

### How to Copy it for Your Own Computer

1.  **Clone the Repository:**
    Open your terminal or command prompt and clone the project:
    ```bash
    git clone https://github.com/cooleditingx/Odin_Dashboard.git
    cd Odin_Dashboard
    ```

2.  **Open and View:**
    Simply open the primary file, `index.html`, in your web browser.
    ```bash
    open index.html # For macOS/Linux
    # OR
    start index.html # For Windows
    ```

### How to Use it

* Navigate the layout and inspect the HTML and CSS code to see the structure and styling implementation.
* Resize your browser window to see the dashboard's **responsive design** in action.

## ‚ú® Features

The project incorporates several key structural and design features:

* **Grid-Based Layout:** Utilizes CSS Grid for the main dashboard structure, ensuring a robust and organized content flow.
* **Persistent Sidebar Navigation:** A clean, visually distinct sidebar for navigation, consistent across the application.
* **Header with Search Functionality:** Includes a header area for user profile, announcements, and a search bar.
* **Dynamic Project Cards:** The main content area features visually engaging cards for displaying various projects, suitable for portfolio showcases.
* **Responsive Design:** Optimized to maintain functionality and aesthetics across various screen sizes, from desktop to mobile.
